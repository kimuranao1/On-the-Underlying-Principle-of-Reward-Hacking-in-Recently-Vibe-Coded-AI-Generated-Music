Statistical Analysis of Quintet Relative Phase Melodies

This analysis focuses on classical MIDI works, extracting melodies as relative phase motion sequences (quintet structures) rather than pitches, and analyzing their statistical distribution.

The analysis target is not sound, but "motion shape."

Analysis Conditions

  • Converting Simultaneous Sounds to Monophonic Sounds
  • Automatic Tonality Estimation
  • Tonic Normalization
  • Pitch Difference Extraction
  • 5-Consecutive Structure Extraction
  • Relative Phase Normalization

This process eliminates all of the following:

Pitch, Key, Composer, and Period

Only pure structures are compared.
